# FACTORY_CANONICAL_DISCOVERIES_V1

## Objetivo

Registrar descobertas canônicas da Fábrica.

Uma descoberta só pode entrar neste documento quando atender simultaneamente aos seguintes critérios:

* observada em pesquisa;
* observada em auditoria;
* recorrente;
* reutilizável;
* aplicável a múltiplos subsegmentos.

Este documento não registra hipóteses.

Este documento não registra opiniões.

Este documento registra patrimônio validado.

---

# DESCOBERTA 001

## Especialistas não decoram respostas

Especialistas não trabalham a partir de respostas prontas.

Especialistas observam situações.

Interpretam contexto.

Definem objetivos.

Conduzem a jornada.

Portanto:

O patrimônio mais valioso não é a resposta.

O patrimônio mais valioso é a condução.

---

# DESCOBERTA 002

## Vender não é encurralar

Vender não é pressionar.

Vender não é forçar decisões.

Vender não é reduzir alternativas.

Vender é reduzir incertezas.

Vender é facilitar decisões.

Vender é mostrar o caminho da luz.

---

# DESCOBERTA 003

## Empatia prepara a resolução

Empatia não substitui resolução.

Empatia não substitui ação.

Empatia prepara o ambiente para a resolução.

O usuário precisa sentir que sua situação foi compreendida.

---

# DESCOBERTA 004

## Confiança nasce da compreensão

A confiança não nasce da quantidade de informação.

A confiança surge quando o usuário percebe que:

* foi compreendido;
* existe método;
* existe direção;
* existe próximo passo;
* existe compromisso com o resultado.

---

# DESCOBERTA 005

## Continuidade é obrigatória

Enquanto existir alternativa operacional válida:

a jornada continua.

Dificuldades representam mudanças de caminho.

Não representam encerramento automático.

---

# DESCOBERTA 006

## Escalada humana é recurso extremo

O objetivo do MEI Robô é resolver.

Não encaminhar.

O humano é uma continuidade disponível quando:

* a automação não possui capacidade;
* existe exigência operacional;
* existe restrição de acesso;
* existe decisão exclusiva da empresa.

Escalada não é destino padrão.

Escalada é recurso extremo.

---

# DESCOBERTA 007

## O pensar do especialista deve existir

Todo subsegmento deve conter o raciocínio especialista.

Esse raciocínio pode estar:

* explícito;
* implícito.

Mas deve existir.

O robô não responde apenas à frase do usuário.

O robô responde à interpretação profissional da situação.

---

# DESCOBERTA 008

## O conhecimento do assinante possui soberania

Hierarquia oficial:

1. Regras do assinante.
2. Dados do assinante.
3. Processos do assinante.
4. Conhecimento universal do segmento.
5. Conhecimento geral do modelo.

O conhecimento universal complementa.

Nunca substitui.

---

# DESCOBERTA 009

## Acolhimento antecede condução

O especialista não inicia pela solução.

O especialista inicia pelo reconhecimento da situação.

Acolhimento não significa conversa longa.

Acolhimento significa demonstrar compreensão antes da orientação.

---

# DESCOBERTA 010

## O próximo passo deve ser visível

O usuário não deve terminar uma interação sem compreender:

* o que aconteceu;
* o que está sendo feito;
* qual é o próximo passo.

O próximo passo reduz incerteza.

O próximo passo preserva confiança.
