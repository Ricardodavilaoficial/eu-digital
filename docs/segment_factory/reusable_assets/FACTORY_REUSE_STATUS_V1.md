# FACTORY_REUSE_STATUS_V1

## Objetivo

Registrar rapidamente o estado atual dos mecanismos reutilizáveis da Fábrica.

Este arquivo existe para acelerar novos subsegmentos.

Antes de pesquisar um novo segmento:

1. consultar este arquivo;
2. identificar mecanismos reutilizáveis;
3. identificar candidatos pendentes;
4. pesquisar apenas o que ainda não foi validado.

# Observação

Componentes não substituem pesquisa.

Componentes reduzem retrabalho.

- reutilizar o que já foi validado;
- validar candidatos;
- promover somente após recorrência observável.
