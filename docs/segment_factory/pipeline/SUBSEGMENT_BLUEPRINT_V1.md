# SUBSEGMENT BLUEPRINT V1

## Objetivo

Criar um padrão único para construção de subsegmentos do MEI ROBÔ.

O objetivo não é armazenar respostas prontas.

O objetivo é armazenar raciocínios que permitam ao GPT-4o-mini responder como um profissional experiente.

---

# Princípio central

A KB não responde pelo robô.

A IA responde usando a KB.

Portanto:

* evitar scripts;
* evitar respostas prontas;
* evitar FAQs gigantes;
* evitar abstrações excessivas;
* evitar conceitos sem aplicação prática.

Priorizar:

* raciocínio profissional;
* interpretação de comportamento;
* critérios de decisão;
* direção segura de resposta.

---

# CAMADA 1 — OPERAÇÃO

Objetivo:

Como o negócio funciona.

Campos:

* operational_ritual
* operational_rules
* real_customer_situations
* handoff_patterns

---

# CAMADA 2 — EXPERTISE TÉCNICA

Objetivo:

Como o especialista pensa.

Cada tópico deve conter:

* principle
* why_it_matters
* common_customer_confusion
* decision_factors
* safe_response_direction

---

# CAMADA 3 — RACIOCÍNIO PROFISSIONAL

Objetivo:

Como o especialista toma decisões.

Campos:

* professional_reasoning_patterns
* expert_reframing_patterns

Exemplo:

Cliente:
"Qual lente é melhor?"

Reframe do especialista:

"Melhor para qual rotina de uso?"

---

# CAMADA 4 — COMPORTAMENTO DO CLIENTE

Objetivo:

Interpretar o que o cliente realmente quer dizer.

Campos:

* behavioral_clusters

Estrutura:

* cluster_name
* observable_patterns
* most_likely_meaning
* recommended_direction
* avoid

---

# CAMADA 5 — PSICOLOGIA DE COMPRA

Objetivo:

Compreender os fatores humanos envolvidos na decisão.

Campos:

* buying_motivations
* fear_drivers
* decision_delay_patterns

---

# CAMADA 6 — CONSTRUÇÃO DE CONFIANÇA

Objetivo:

Gerar confiança progressiva.

Campos:

* trust_progression_patterns
* trust_building_patterns
* trust_breaking_patterns

Princípio:

O vendedor especializado deve construir confiança enquanto conduz a conversa.

---

# CAMADA 7 — ESTRATÉGIA COMERCIAL

Objetivo:

Conduzir sem pressionar.

Campos:

* sales_principles
* value_perception_patterns

Evitar:

* urgência falsa;
* medo artificial;
* promessas absolutas;
* manipulação.

---

# CAMADA 8 — CUSTOMIZAÇÃO DO ASSINANTE

Objetivo:

Permitir diferenciação entre empresas.

Campos futuros:

* custom_expertise
* custom_sales_style
* custom_differentials
* custom_policies
* promotions
* brands
* guarantees

---

# Regra para GPT-4o-mini

Todo conhecimento armazenado deve responder:

1. O que o especialista sabe?
2. O que o cliente quer dizer?
3. Qual direção de resposta é mais adequada?

Evitar:

* conceitos abstratos;
* listas genéricas;
* textos enciclopédicos.

Priorizar:

* padrões observáveis;
* critérios de decisão;
* interpretações acionáveis.
