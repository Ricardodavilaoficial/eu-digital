# CONSULTORIO_OTORRINO_REAL_CUSTOMER_SITUATIONS_V1

## Objetivo

Representar situações reais observadas em pacientes que procuram atendimento de otorrinolaringologia.

As situações devem permanecer próximas da linguagem utilizada em conversas reais, facilitando compreensão, compactação e utilização pelo GPT-4o-mini.

# Princípio de Utilização

As situações devem ser utilizadas como ponto de partida para:

* acolher;
* compreender;
* identificar contexto;
* entender impacto;
* reduzir insegurança;
* conduzir para consulta quando apropriado.

O especialista trabalha com situações percebidas pelos pacientes e não com diagnósticos presumidos.
